How to play:

How to play

When you press "Enter" or "Space", the game will start.

The snake will move forward to the direction it look at.

You can press "WASD" or "^v<>" to turn snake head.

If the snake hit it's body or the map boundary, the snake will dead immediately

How to compile:

1.Download the source code:

git clone

2.Install cmake:

cmake

3.Download SFML 2.5.1:

SFML

4.Build SFML library by the toturial in SFML website

Toturial

5.Edit the cmake setting file in snakeGame/CMakeLists.txt, change the SFML_DIR to your sfml build directory.
